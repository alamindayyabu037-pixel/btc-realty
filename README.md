BTC Realty is a modern real estate digital network built to bridge the gap between 
property owners, buyers, and renters across Nigeria. The platform allows founders 
to list verified properties — from luxury villas and apartments to land and shortlets 
— while giving buyers and renters a seamless, elegant experience to search, filter, 
and connect with agents instantly via WhatsApp, call, or inquiry form.

Built with a clean HTML, CSS, and JavaScript frontend, with a PHP & MySQL backend 
coming soon for full property management, real-time inquiries, and admin control.
